Activity Index¶

Activities (alphabetical)¶

Add Non Capital Alternatives

Activity that adds a lower-case text variant to stretches of text that are in all-caps

Annotate Homonyms

The activity AnnotateHomonyms annotates locations that host multiple terms [effectively: homonyms] for further processing.

Annotate Patterns

Pattern annotation activity

References static objects:

  • Type PatternAnnotator, specify name by Annotator

Annotate P O S Patterns

Pattern annotation activity that annotates patterns POS tags.

References static objects:

  • Type POSPatternMatcher, specify name by Annotator

Annotate Terms

The activity AnnotateTerms flags terms based on concept id lists or term id lists.

References static objects:

  • Type Attributes, specify name by ConceptAttributes (Optional)
  • Type Disambiguator, specify name by TermAnnotator (Optional)
  • Type Attributes, specify name by TermAttributes (Optional)
  • Type TokenRangeMatcher, specify name by ContextMatcher (Optional)

Annotate Token Range Patterns

Pattern annotation activity based on token ranges [rather than plain input text]. Produces RangeAnnotations.

References static objects:

  • Type TokenRangeMatcher, specify name by Annotator
  • Type Normalizer, specify name by Normalizer (Optional)

Annotate Token Ranges

Pattern annotation activity based on token ranges [rather than plain input text]. Produces TokenPatterns.

References static objects:

  • Type TokenRangeMatcher, specify name by Annotator

Dehyphenate

The Dehyphenate activity looks for alpha-hyphen-whitespace-alpha (A-WA) patterns and tries to reconstruct whole words if we can confirm that none of the broken-up words are confirmed to exist, while the whole is.

References static objects:

  • Type WordSet, specify name by Words

Detect Abbreviations

The DetectAbbreviations activity provides functionality to detect abbreviations. It requires input with tokens, sentences, text and normalizations filled out.

References static objects:

  • Type Normalizer, specify name by Normalizer
  • Type AbbreviationSettings, specify name by AbbreviationSettings (Optional)

Detect Capitalisation Regions

The DetectCapitalisationRegions activity detects regions in the text with special capitalisations.

Detect Sentences

DetectSentences activity detects sentences based on the previous tokenization and, optionally, detected patterns.

References static objects:

  • Type SentenceDetector, specify name by SentenceDetector

Distribute Term Flags

The activity DistributeTermFlags distributes flags; either without restriction or based on abbreviation expansions alone.

Expand Abbreviations

Class that defines methods to put found abbreviations to use: expand portions of text that represent an abbreviation to their long forms, and eliminate short forms altogether.

Expand Concept Coordinations

Expand Concept Coordinations activity adds alternative [expanded] readings for a text if there is enough reason to assume, based on an existing set of concepts [thesaurus], that the text contains an elliptic coordination.

References static objects:

  • Type ConceptCoordinationPatternMatcher, specify name by CoordinationPatterns (Optional)
  • Type WordSet, specify name by Coordinators
  • Type Thesaurus, specify name by Thesaurus

Expand Coordinations

Expand coordinations activity expands morphological coordinations such as ‘homo- and heterozygote’ based on a predefined set of coordinating prefixes

References static objects:

  • Type TokenPatternMatcher, specify name by CoordinationPatterns (Optional)
  • Type WordSet, specify name by Coordinators
  • Type PrefixPair, specify name by WordPairs

Find Terms

The activity FindTerms finds terms in the text based on a thesaurus or meta-thesaurus.

References static objects:

  • Type WordSet, specify name by IgnoreTerms (Optional)
  • Type Attributes, specify name by TermPOSTags (Optional)
  • Type Thesaurus, specify name by Thesaurus

Flag Terms

The activity FlagTerms flags terms based on concept id lists or term id lists.

References static objects:

  • Type WordSet, specify name by ConceptsToFlag (Optional)
  • Type WordSet, specify name by TermsToFlag (Optional)

Hide Patterns

The HidePatterns activity adds EmptyExpansion annotations to the annotation storage for TokenPatterns of certain types, effectively hiding the annotated regions of text from further inspection by activities that are sensitive to this type of hiding, for instance FindTerms.

Make Fingerprint

The activity MakeFingerprint aggregates found terms of a certain thesaurus to a concept fingerprint.

References static objects:

  • Type StringMap, specify name by ConceptNameTable (Optional)
  • Type ConceptWeightsPerSection, specify name by ConceptWeightsDefinition (Optional)

Markup Words

MarkupWords adds flags to words contained in a specified WordSet.

References static objects:

  • Type WordSet, specify name by WordSetName

Multi Normalize

The multi-normalize activity inserts MultiTokenWord annotations for known multi-word units.

References static objects:

  • Type MultiNormalizer, specify name by MultiNormalizer

Normalize

Normalize activity

References static objects:

  • Type Normalizer, specify name by Normalizer

Tag Parts Of Speech

TagPartsOfSpeech activity can perform part-of-speech tagging with both Beam and Viterbi engine

References static objects:

  • Type LanguageModel, specify name by Model
  • Type TokenPatternToTagProbabilityDistribution, specify name by TokenPatternTagProbabilities (Optional)

Tokenize

The Tokenize activity splits the text in Tokens, and identifies the Words; that is, for non-whitespace tokens, base Word forms are also inserted.

References static objects:

  • Type Tokenizer, specify name by Tokenizer
